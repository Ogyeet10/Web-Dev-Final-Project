InstantMessager User Manual

How to Use InstantMessager

Step 1: Access the Connect Page

Navigate to the Connect page using the navigation menu or the "Start Messaging" button on the home page.

Step 2: Generate or Enter a Session ID

You can either:

  • Generate a new ID: Click the "Generate ID" button to create a unique session identifier
  • Enter an existing ID: Type in a session ID that someone has shared with you
Step 3: Share Your ID

Copy your session ID and share it with the person you want to chat with. They need to enter this ID on their Connect page.

Step 4: Start Messaging

Once both users have entered the same session ID, you'll be connected and can start sending messages in real-time.

Features & Tips

Real-time Messaging

Messages appear instantly without needing to refresh the page.

Session-based

No account registration required. Just share session IDs to connect.

Privacy

Each session is temporary and private to the users who share the ID.

Troubleshooting

  • Can't connect? Make sure both users are using the exact same session ID
  • Messages not sending? Check your internet connection and try refreshing the page
  • Lost your session? Generate a new ID and share it again
